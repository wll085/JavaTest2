# 第二次测试
## 使用说明
  - 执行命令 $ git clone https://github.com/wll085/JavaTest2.git
  - sh build.sh
  - build.sh执行完成后已经重新打包
  - 执行Exam1的程序
    - cd Exam1
    - sh run.sh
  - 执行Exam2的程序
    - cd Exam1
    - 先启动服务器端执行java -cp target/Exam2.jar TCPServer
    - 再另外开一个控制台执行命令java -cp target/Exam2.jar TCPClient
